# 自定义图标
weex内置模块dom
- addRule
```
onLaunch() {
	const domModule = weex.requireModule('dom')
	domModule.addRule('fontFace', {
	    'fontFamily': "iconfont",
	    'src': "url('http://at.alicdn.com/t/font_1469606063_76593.ttf')"
	});
}
```
1. 图标写在text组件里
2. 组件能不能换行
